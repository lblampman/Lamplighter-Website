HTML5 Markup

HTML5 is used for structuring and presenting content on the World Wide Web. It is the fifth and current version of the HTML standard and introduced elements and attributes that reflect typical usage on modern websites. This is the version used by Lamplighter Web Design.

CSS3 Styling

CSS3 is the third iteration of Cascading Style Sheets, which is used most often to set the visual style of web pages. CSS is a cornerstone technology in creating visually engaging websites. This is the version used by Lamplighter Web Design.

Graphic Design

The actual visual layout and design from which we create the website using HTML5 and CSS3. It also encompasses the important underlying "flow" of the website, whether to enable better workflow in an application, or draw a viewer's attention to key areas of the website.
